# Debate Scoring Rubric

## Dimensions

### Evidence Quality (0-10)
Measures whether cited data is real, sufficient, and reproducible.

| Score | Description |
|-------|-------------|
| 0 | No evidence cited |
| 1-2 | Anecdotal or irrelevant evidence |
| 3-4 | Some evidence but with significant gaps or questionable sources |
| 5-6 | Adequate evidence from the paper, but not comprehensive |
| 7-8 | Strong evidence with specific citations, reproducible methodology |
| 9-10 | Comprehensive, reproducible evidence with statistical rigor |

### Logical Validity (0-10)
Measures whether the conclusion follows from the premises.

| Score | Description |
|-------|-------------|
| 0 | Non-sequitur — conclusion unrelated to premises |
| 1-2 | Major logical gaps or fallacies |
| 3-4 | Some logical connection but with significant leaps |
| 5-6 | Reasonable chain of reasoning with minor gaps |
| 7-8 | Strong logical chain with well-supported premises |
| 9-10 | Airtight logical chain, all premises supported, no gaps |

### Counterargument Handling (0-10)
Measures how well the agent addressed opposing evidence.

| Score | Description |
|-------|-------------|
| 0 | Completely ignored opposing evidence |
| 1-2 | Acknowledged opposition but didn't address it |
| 3-4 | Weak or deflective response to counterarguments |
| 5-6 | Addressed main counterarguments but missed some |
| 7-8 | Thoroughly addressed counterarguments with evidence |
| 9-10 | Decisively rebutted all counterarguments with superior evidence |

### Novelty/Insight (0-5)
Measures non-obvious contributions to understanding.

| Score | Description |
|-------|-------------|
| 0 | Restates commonly known information |
| 1 | Minor new angle on known information |
| 2-3 | Meaningful new perspective or connection |
| 4-5 | Genuine breakthrough insight or novel synthesis |

### Credibility Weight (0-5)
Based on the paper's empirical track record (from credibility profile).

| Score | Description |
|-------|-------------|
| 0 | Paper retracted, no reproducibility, no citations |
| 1 | Preprint only, no code/data, minimal citations |
| 2 | Preprint with some reproducibility signals |
| 3 | Peer-reviewed, moderate citations and reproducibility |
| 4 | Top venue, good citations, code + data available |
| 5 | Top venue, highly cited, fully reproducible, strong author track record |

## Total Score
Maximum raw score: 40 points (10+10+10+5+5)
Penalties are subtracted from the raw score.
Minimum adjusted score: 0 (no negative totals).
